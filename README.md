# LešenářiBugáň

Website for BUGÁŇ MONTÁŽE A STAVBY s.r.o. - professional scaffolding assembly and rental services in Czech Republic and Slovakia.

**Live site:** https://juraj0137.github.io/lesenaribugan/

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- Static export for GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
